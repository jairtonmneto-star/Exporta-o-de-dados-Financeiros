#Dashboard de Análise de Criptomoedas
Esse e um Projeto No qual eu usei o Vs code a linguagem python e o pacote yfinance,pandas,streamlit. para fazer um Dashboard de Analise de Criptomoedas de 4 marcas  BTC-USD = Bitcoin Dos Estados Unidos, ETH-USD = Ethereum dos Estado Unidos, LTC-USD = Litecoin dos Estados Unidos, XRP-USD = Ripple dos Estados Unidos
Aonde voce tem que baixar o yfinance no site
e depois de usar os dados Rodar com python -m streamlit run nome do arquivo.py

Como Instalar o yfinance:
pip install yfinance

📌 Funcionalidades

KPIs em tempo real: preço atual, máximo/mínimo 52 semanas e variação diária
Gráfico de linha interativo com filtro por período
Comparação entre múltiplas criptomoedas simultaneamente
Tabela com dados históricos
